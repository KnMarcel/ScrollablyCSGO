# ScrollablyCSGO

## Folder structure
```
ScrollablyCSGO/
├── index.html              ← the entire dashboard (open this in browser)
├── precompute_dashboard.py ← run once locally to generate JSON files
├── maps/                   ← put your PNG map images here manually
│   ├── de_dust2.png
│   ├── de_mirage.png
│   ├── de_inferno.png
│   ├── de_overpass.png
│   ├── de_cache.png
│   ├── de_cbble.png
│   └── de_train.png
└── dashboard/              ← generated by precompute_dashboard.py
    ├── de_dust2.json
    ├── de_mirage.json
    └── ...
```

## Setup
1. Copy your map PNGs into the `maps/` folder
2. Run: `python precompute_dashboard.py`
3. Upload the `dashboard/*.json` files to HuggingFace Dataset `HeadShottt/CS-GO` under a `dashboard/` folder
4. Open `index.html` in your browser (or host on GitHub Pages)

## GitHub Pages
Put this entire folder in your repo under `docs/ScrollablyCSGO/` and enable GitHub Pages on the `docs/` folder.
